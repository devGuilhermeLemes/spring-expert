# spring-expert

## 1° Parte - Produtos API – Spring Boot

Este projeto é uma API REST desenvolvida com Spring Boot, com foco em boas práticas de organização, arquitetura em camadas e padronização de endpoints HTTP.

Este repositório corresponde à 1ª parte do projeto, onde foi criada a base da aplicação e as primeiras requisições REST.

Tecnologias utilizadas

- Java

- Spring Boot

- Spring Web (REST)

- Spring Data JPA

- Maven

- YAML (application.yml)

- Git / GitHub

O objetivo desta primeira etapa é construir a estrutura inicial de uma API REST, aplicando conceitos fundamentais como:

- Controllers REST

- Separação de responsabilidades (Controller / Model / Repository)

- Comunicação via HTTP

- Retorno de dados em JSON

- Organização profissional de pacotes
